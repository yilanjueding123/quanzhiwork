	本目录（pub1)下的文件由Gary创建并维护，其他人只有读权限。
	这些文件各自完成一些通用的功能，会被多方使用，故独立出来集中放置。

	目前为止，此目录下包含如下文件：
1. type_def.h
2. G_debug.h
3. G_os_related.h
4. G_os_related.c


1. type_def.h
--------------------------------------------------------------------------------
	此头文件中定义了一些基本的数据类型，常量和宏函数。
	
	
	
2. G_debug.h
--------------------------------------------------------------------------------
	此头文件定义了调试打印的级别。
	
	
3. G_os_related.h
--------------------------------------------------------------------------------
	此文件和G_os_related.c文件一起共同定义了与操作系统相关的接口，使得代码能够在
不同的平台上实现平滑地切换。


4. G_os_related.c
--------------------------------------------------------------------------------
	此文件和G_os_related.h文件一起共同定义了与操作系统相关的接口，使得代码能够在
不同的平台上实现平滑地切换。
