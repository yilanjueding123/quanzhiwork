
//--------------------------------------------------------------------
            MINFS文件系统PC工具使用说明
估算镜像空间
minfs calc  source
source  用于估算镜像文件尺寸信息的源目录

创建镜像文件
minfs make source dest config
source  创建镜像文件的源目录
dest    镜像文件文件路径
config  镜像创建的配置文件路径

导出镜像文件
minfs dump souce dest
source  镜像文件文件路径
dest    镜像文件导出后存放的目标目录
