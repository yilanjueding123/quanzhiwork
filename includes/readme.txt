/**********************************************************************************************************************/
/* 1.0 使用说明                                                                                                       */
/**********************************************************************************************************************/

1.1     用户若需要编写应用或驱动,请包含epdk.h
1.2     用户若需要独立使用ebios去移植其他的系统,请包含ebios.h
1.3     宏定义
        EPDK_CPU        定义cpu的版本，比如EPDK_CPU_AC320,       EPDK_CPU_ARM920T,    EPDK_CPU_IDU
        EPDK_ARCH       定义产品结构版本，比如EPDK_ARCH_SUN
        EPDK_CHIP       定义产品版本，比如EPDK_CHIP_SUNI, EPDK_CHIP_SUNII
        EPDK_OS         定义操作系统版本，比如EPDK_OS_EPOS,         EPDK_OS_WIN2K,       EPDK_OS_WINXP
        DEBUGLEVEL      定义调试级别，比如
        
        所有的C常量定义请参考“includes\const.h” 与 “includes\dgs.h”
        
        在使用epdk.h头文件前需要预先定义这些宏到某个具体的变量上，否则编译或出错，定义这些宏可以在自己的总头文件里先定义，
        然后包含 "epdk.h" 
      
        #define EPDK_CPU    EPDK_CPU_ARM920T
        #define EPDK_ARCH   EPDK_ARCH_SUN
        #define EPDK_CHIP   EPDK_CHIP_SUNI
        #define EPDK_OS     EPDK_OS_EPOS
        #define DEBUGLEVEL  EPDK_DEBUG_LEVEL_LOG_ALL
        #include "ePDK.h"



        对于高级用户，也可以在工程文件里直接包含"includes\\cfgs\\..."的via文件，via文件具体使用请参考相关手册
